manual

  1. +1 kg button:

    This is a must-have item for match anglers. There are usually limits about how much weight can be in your keep nets during matches. They usually use a clicker counter to control the weight amount when counting roughly. Examine how the +1 kg button operates.

  2. catch counter:

    Simply counting the number of fish caught and assisting in the control of the fishing area.

  3. stopwatch:

    A stopwatch is essential for finding a rhythm to the bites you're getting.Aiding in the analysis of bite timing and so making it easier for anglers to count the rebait time.

  4. catch times:

    The list of catch timings helps to simplify the differences in catch times. Displaying the actual time difference between catches.

  5. notes:

    Using match notes and, for example, a lake, could be useful in practising and rematching.
